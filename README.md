# nodejs-web-express-api-ssl-hello-world

## Description
A expressjs api that uses self signed ssl.

Features: 
- self signed ssl

## Tech stack
- expressjs

## Docker stack
- alpine:edge
- node:latest

## To run
`sudo ./install.sh -u`
- curl -i -k https://localhost/

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
